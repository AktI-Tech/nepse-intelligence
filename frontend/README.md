# NEPSE Intelligence Frontend

React 18 + TypeScript frontend for NEPSE Intelligence trading dashboard.

## Features

- Live market data display
- Top gainers/losers
- Stock price tracking
- Real-time updates (via React Query)
- Responsive design (dark theme)

## Development

```bash
npm install
npm run dev
```

App runs on `http://localhost:3000` with automatic proxy to backend API on `http://localhost:8000`.

## Build

```bash
npm run build
```

Builds optimized production bundle in `dist/` directory.

## Technologies

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Fast dev server & build
- **React Query** - Data fetching & caching
- **React Router** - Routing
- **Axios** - HTTP client
